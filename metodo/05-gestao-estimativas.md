# 05 — Gestão e Estimativas: Pessoas > Métricas

## Princípios de gestão

1. **Não existe metodologia mágica** — Processos devem ser customizados para cada projeto
2. **Pessoas > Métricas** — Entender gente é mais importante que dashboards
3. **Métricas de produtividade são perigosas** — Linhas de código e tickets fechados iludem mais do que informam
4. **Estimativas são hipóteses** — Ordens de grandeza, não contratos
5. **Priorização = exclusão** — O mais importante primeiro, o resto no fim da lista
6. **Líder = Firewall** — Protege a equipe das ansiedades externas

## O problema do escopo

> "Se aumentar o escopo, ou o custo aumenta, ou o tempo aumenta. Não existe outra forma."

O escopo é sempre mal definido no início. Só durante o projeto o entendimento melhora. Por isso, revisão constante é obrigatória.

# Como Contribuir

Obrigado por considerar contribuir! Este é um projeto da comunidade e toda ajuda é bem-vinda.

---

## Formas de contribuir

### 🐛 Encontrou um erro?
Abra uma [Issue](https://github.com/johnnyhelder/claude-code-super-power/issues) descrevendo:
- Onde está o erro (qual ficheiro, qual seção)
- O que está errado
- O que deveria estar correto

### 💡 Tem uma sugestão?
Abra uma Issue com a tag `sugestão` explicando:
- O que você gostaria de ver
- Por que seria útil
- Se possível, como implementar

### 📝 Quer melhorar um documento?
1. Faça um Fork do repositório
2. Crie uma branch (`git checkout -b melhoria/nome-da-melhoria`)
3. Faça suas alterações
4. Commit (`git commit -m 'Melhoria: descrição curta'`)
5. Push (`git push origin melhoria/nome-da-melhoria`)
6. Abra um Pull Request

### 🔧 Quer adicionar templates para uma nova stack?
Adoramos isso! Se você usa uma stack diferente (Python, Go, PHP, etc.) e quer criar templates de testes ou CI/CD, siga o processo de PR acima.

### 📖 Quer traduzir?
Se quiser traduzir para inglês ou outro idioma, abra uma Issue primeiro para alinharmos a estrutura.

### 🎯 Completou o Desafio dos 7 Dias?
Compartilhe sua experiência! Abra uma Issue com a tag `experiência` contando:
- Que projeto fez
- O que funcionou bem
- O que ajustou
- O que aprendeu

---

## Regras simples

1. **Mantenha o tom acolhedor** — Este repositório é para iniciantes também
2. **Mantenha os créditos** — Sempre que referenciar conteúdo de terceiros, cite a fonte
3. **PT-BR como idioma principal** — Documentos em português brasileiro
4. **Seja construtivo** — Críticas são bem-vindas quando acompanhadas de sugestões
